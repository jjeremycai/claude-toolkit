# CTO Audit Plugin

Performs deep, expert-level codebase and architecture audits to surface strengths, risks, and opportunities.

## What It Does

- Reviews architecture, implementation quality, performance, and security
- Flags correctness issues, scaling risks, and operational gaps
- Produces prioritized, actionable recommendations

## Usage

```
"Audit this repo for scalability and reliability risks"
"Give a CTO-level assessment of our backend architecture"
"Review the codebase for performance bottlenecks and tech debt"
```

## Outputs

- Summary of system purpose, strengths, and top risks
- Findings grouped by severity with evidence
- Prioritized recommendations and overall assessment
