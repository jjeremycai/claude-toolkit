# Frontend Design Plugin

Generates distinctive, production-grade frontend interfaces that avoid generic AI aesthetics.

## What It Does

- Chooses a clear aesthetic direction and expressive typography
- Builds production-ready UI code with strong visual hierarchy
- Uses purposeful motion and layout details

## Usage

```
"Create a landing page for an AI security startup"
"Build a dashboard for a music streaming app"
"Design a settings panel with dark mode"
```
